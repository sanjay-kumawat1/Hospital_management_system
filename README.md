Hospital Management System

A modern and responsive Hospital Management System developed using Python Full Stack Web Development technologies. This project helps manage hospital operations digitally, including patient records, doctor management, appointments, billing, and admin controls.

🚀 Features

	👨‍⚕️ Doctor Management

	🧑‍🤝‍🧑 Patient Registration System

	📅 Appointment Booking

	🧾 Billing Management

	🔐 Admin Login Authentication

	📊 Dashboard with Hospital Statistics

	📱 Responsive User Interface
